## ติดตั้งและใช้งาน
* แก้ไขเบอร์ที่ต้องการยิงในไฟล์ config.txt ก่อน
```bash
#เบอร์ที่ต้องการยิง

phone = 0999999999
```
* ก่อนใช้งานต้องติดตั้ง library
```bash
pip3 install -r requirements.txt
```
* ใช้งาน ZXVXZV-SMS
```bash
python3 main.py
```
* หากต้องการหยุดยิงให้กด Ctrl + C
